HTML & CSS Basics Summary

    HTML Summary

    HTML is used to define our page content, structure and meaning. You don't use it for styling purposes. Use CSS for that instead.

  1. html uses "elements" to describe(annotate) content

    2. html elements typically have an opening tag, content and then a closing tag

  2. You can also have void(empty) elements like images

  3. You can also configure elements with attributes

    4. there's a long ling of available elements but you'll gain experience over time, no worries.

    Lean more about all available html elements on the MDN HTML element reference.

    CSS Summary

    CSS (Cascading Style Sheets) is used for styling your page content.

    Styles are assigned via property-value pairs

    You can assign styles via the "style" attribute

  4. To avoid cod duplication, you typically use global s tyle (e.g via the "style" element) instead

    5. Alternatively, you can work with external stylesheet files which you "link" to

  5. When working with CSS, concepts like "inheritance", "specificity" and the "box model" should be understood.

    6. Learn more about all available CSS properties and values on the MDN CSS property reference.
